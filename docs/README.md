Thư mục chứa tài liệu kỹ thuật cho STM32F411RE.

## Tài Liệu Cần Tải

Tải miễn phí tại [st.com](https://www.st.com/en/microcontrollers-microprocessors/stm32f411re.html#documentation)

| File | Mô tả | Dùng từ bài |
|---|---|---|
| `RM0383_STM32F411.pdf` | **Reference Manual** — nguồn chính, mô tả toàn bộ thanh ghi | L1 |
| `PM0214_Cortex-M4.pdf` | **Programming Manual** — SysTick, NVIC, MPU, FPU | L2 |
| `stm32f411re_datasheet.pdf` | **Datasheet** — pinout, AF table, electrical specs | L1 |
| `STM32F411.svd` | **SVD file** — dùng với Cortex-Debug để xem Peripherals panel | L1 |
| `AN4365_DMA.pdf` | App Note: Getting started with DMA | L6 |
| `AN2606_BootMode.pdf` | App Note: STM32 system memory boot modes | L8 |
